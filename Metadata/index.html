📝 Project Description

NH Token is a decentralized, community-driven Web3 commerce token built on the Polygon (PoS) network.

It enables:

  • Instant, KYC-compliant e-commerce payments

  • 0.5% of every transaction routed into liquidity — building sustainable on-chain ecosystems

  • Fully self-funded, founder-developed, and publicly auditable on GitHub

🔍 Use Cases:

  • KYC-verified Web3 payments (focus: Switzerland & India)

  • Smart contract-based payment plugins

  • Dynamic DeFi integrations (NFTs, staking, services)

  • Whitelist-based compliance using Merkle Tree

⭐ Key Features:

  • Built on Polygon Mainnet

  • Auto-liquidity: 0.5% per transaction

  • No VC funding / no early unlocks

  • Web3 payment gateway-ready
